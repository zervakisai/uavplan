## 🚀 How to Save Videos

### Command 1: Save Best Path
```bash
uavbench --scenarios urban_easy --planners astar --trials 5 --save-videos best
```
**Result**: Saves the best (shortest) successful path as a video
- **File**: `videos/urban_easy_astar_best_seed<SEED>.gif`

### Command 2: Save Worst Path
```bash
uavbench --scenarios urban_easy --planners astar --trials 5 --save-videos worst
```
**Result**: Saves the worst (longest) successful path as a video
- **File**: `videos/urban_easy_astar_worst_seed<SEED>.gif`

### Command 3: Save Both Best and Worst
```bash
uavbench --scenarios urban_easy --planners astar --trials 5 --save-videos both
```
**Result**: Saves both best and worst paths
- **Files**:
  - `videos/urban_easy_astar_best_seed<SEED>.gif`
  - `videos/urban_easy_astar_worst_seed<SEED>.gif`

### Command 4: Save with Custom FPS
```bash
uavbench --scenarios urban_easy --planners astar --trials 5 --save-videos best --fps 15
```
**Result**: Saves video with custom playback speed (default is 8 FPS)

## 📊 Video Details

### File Format
- **Default**: GIF (works without ffmpeg)
- **With ffmpeg installed**: MP4 (compressed, smaller file size)
- If MP4 fails, automatically falls back to GIF ✓

### Video Content
- **Grid**: Gray cells = buildings, red cells = no-fly zones
- **Path**: Blue line showing the planned trajectory
- **Start**: Green dot with dark green border
- **Goal**: Gold star with orange border
- **Animation**: Frame-by-frame movement from start to goal

### File Size
- Typical GIF: 40-100 KB (depending on path length and complexity)
- MP4 (if ffmpeg installed): 10-30 KB (smaller due to compression)

## 🛠️ Troubleshooting

### Videos Not Appearing
1. **Check CLI output** for error messages
   ```bash
   uavbench --scenarios urban_easy --planners astar --trials 2 --save-videos best 2>&1
   ```

2. **Verify videos folder exists**
   ```bash
   mkdir -p videos
   ```

3. **Check Pillow is installed** (for GIF export)
   ```bash
   pip install Pillow
   ```

### MP4 Not Saving (Falls Back to GIF)
- MP4 requires ffmpeg to be installed on your system
- This is **optional** - GIF format works fine without it
- To enable MP4:
  ```bash
  # macOS
  brew install ffmpeg
  
  # Ubuntu/Debian
  sudo apt-get install ffmpeg
  
  # Fedora
  sudo dnf install ffmpeg
  ```

### Videos Folder Full (Too Many Files)
```bash
# Remove all videos
rm videos/*.gif
rm videos/*.mp4

# Or remove old videos
find videos/ -name "*.gif" -mtime +7 -delete  # Files older than 7 days
```

## 🔧 Technical Details

### How Videos Are Created
1. **Run benchmark**: Execute planner algorithm and capture path
2. **Detect best/worst**: Select path by length
3. **Create animation**: Generate frame-by-frame visualization
4. **Save video**: Output as MP4 (if ffmpeg available) or GIF (Pillow)
5. **Auto-create folder**: `videos/` directory created if missing

### Implementation
- **File**: `src/uavbench/cli/benchmark.py` (lines 270-280)
- **Save function**: `src/uavbench/viz/player.py::save_path_video()`
- **Path resolution**: Uses `Path` from `pathlib` for cross-platform support

### Code Example
```python
from pathlib import Path
from uavbench.viz.player import save_path_video

# Create videos folder if needed
video_dir = Path("videos")
video_dir.mkdir(parents=True, exist_ok=True)

# Save video
save_path_video(
    heightmap,
    no_fly,
    start,
    goal,
    path,
    video_dir / "my_path.mp4",
    fps=10
)
```
